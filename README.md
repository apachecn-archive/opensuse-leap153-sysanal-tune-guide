## 下载

### Docker

```
docker pull apachecn0/opensuse-leap153-sysanal-tune-guide
docker run -tid -p <port>:80 apachecn0/opensuse-leap153-sysanal-tune-guide
# 访问 http://localhost:{port} 查看文档
```
